Request Capture Context

Once you are ready to render your checkout page or "pay now" button, you need to request the Capture Context from Microform. The capture context request shown below drives Microform's behavior, including accepted card networks, the website(s) where the JavaScript library will be hosted (which acts as a security function), and the version of Microform you're requesting. Once sent to CyberSource, it will produce JWT that can be used to initialize Microform.
